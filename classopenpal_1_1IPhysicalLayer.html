virtual void openpal::IPhysicalLayer::BeginClose ( )
pure virtual

Starts a close operation.

Callback is IHandler::OnLowerLayerDown. Callback occurs after all hronous operations have occured. If the user code has an outstanding read or write, those handlers will not be called.
