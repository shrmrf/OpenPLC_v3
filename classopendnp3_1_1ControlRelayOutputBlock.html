Detailed Description

Describes an incoming control request from the master. It is the applications responsibility to handle the request and return an approiate status code.The PULSE_CLOSE and PULSE_TRIP ControlCodes require setting the mOnTimeMS, mOffTimeMS and mCount variables, otherwise just use the defaults.
